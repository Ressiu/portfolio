JavaScript (JS): É uma linguagem de programação utilizada para tornar as páginas da web interativas e dinâmicas, adicionando funcionalidades como animações, validações de formulários e manipulação de eventos.

CSS (Cascading Style Sheets): É uma linguagem de estilo usada para estilizar a apresentação de elementos HTML em uma página da web, como layout, cores e fontes.

HTML (Hypertext Markup Language): É uma linguagem de marcação usada para criar a estrutura e o conteúdo básico de uma página da web.
